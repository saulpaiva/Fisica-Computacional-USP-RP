# O que é um sitema dinâmico?

- 'Algo' que evoluirá com o tempo
- Regra específica que esse 'algo' evolui no tempo
- 'Algo' deve ser codificado em variáveis
- Regra = funções envolvendo essas variáveis

- Sistema Dinâmico é, então, um modelo que descreve a evolução temporal de um sistema

# Variáveis vs. Funções

# Modelagem de Sistemas Dinâmicos (S.D.)

- Modelos ssão representações simplificadas da realidade que servem para diferentes objetivos:

    - Fazer previsão
    - Descrever padrões
    - Testar hipóteses
    - Sugerir o quê medir, ou quais perguntas fazer (pesquisa exploratória)
    - Analisar dados (Big Data)
    - Revelar mecanismos ou interações
    - Fazer teoria

# SD Contínuos

- Campos de velocidades
- Valores reais
- Normalmente usam EQ Dif's.

# SD Discretos

- Relação de recorrência
- Mapas
- Autômatos Celulares
- Redes Complexas
- Modelos Baseados em Agentes

# Conceitos

- Espaço de estado
    Todos os possíveis estados do sistema
    
- Trajetória ou caminho
    Conjunto de posições no espaço de estado através do qual o sistema eventualmente atravessa. O comportamento do sistema é muitas vezes descrito por trajetórias através do espaço de estado
    
- Atrator
    Ponto do espaço estado a qual o sistema tenderá quando se encontrar na regiao circundante. Esta região circundante é a bacia de atração
